# Creating Pastes

Learn how to create, edit, manage, and share pastes in SecurePaste.

## Creating a New Paste

### Basic Paste Creation

1. Click **New Paste** in the navigation
2. Enter your content in the editor
3. (Optional) Set a title for easy identification
4. Choose syntax highlighting language
5. Click **Create Paste**

### Editor Features

The paste editor includes:

- **Monaco Editor** - The same editor used in VS Code
- **Syntax Highlighting** - 100+ languages supported
- **Line Numbers** - Easy reference and navigation
- **Auto-Indentation** - Smart code formatting
- **Multi-Cursor Editing** - Edit multiple lines simultaneously

### Supported Languages

SecurePaste supports syntax highlighting for 100+ languages including:

- JavaScript, TypeScript, Python, Java, C++, C#
- Go, Rust, Ruby, PHP, Swift, Kotlin
- HTML, CSS, SCSS, JSON, YAML, XML
- Bash, PowerShell, SQL, Markdown
- And many more...

## Paste Settings

### Title

Give your paste a descriptive title to easily identify it later. Titles are optional but recommended for organization.

### Expiration

Set when your paste should automatically delete:

- **Never** - Paste remains until manually deleted
- **1 Hour** - Deleted after 1 hour
- **1 Day** - Deleted after 24 hours
- **1 Week** - Deleted after 7 days
- **1 Month** - Deleted after 30 days
- **Custom** - Set a specific date and time

### Folder/Collection

Organize pastes into folders:

1. Select an existing folder from the dropdown
2. Or create a new folder by typing a name
3. Folders help organize related pastes

## Editing Pastes

To edit an existing paste:

1. Navigate to your paste
2. Click the **Edit** button (only visible for your own pastes)
3. Make your changes
4. Click **Save Changes**

> **Note:** Editing a paste updates it in place. The URL remains the same.

## Cloning/Forking Pastes

To create a copy of any paste:

1. View the paste you want to clone
2. Click **Clone** or **Fork**
3. Make your modifications
4. Save as a new paste

This is useful for:
- Creating variations of existing pastes
- Building on someone else's code
- Keeping a personal copy of important pastes

## Deleting Pastes

To delete a paste:

1. Navigate to your paste
2. Click the **Delete** button
3. Confirm the deletion

> **Warning:** Deletion is permanent and cannot be undone.

## Downloading Pastes

Download any paste as a file:

1. View the paste
2. Click **Download**
3. Choose format (raw text or with metadata)

## Viewing Raw Pastes

For a clean, distraction-free view:

1. Click **Raw** on any paste
2. You'll see the paste content without UI elements
3. Perfect for embedding or quick copying

The raw viewer URL format: `https://raw.securepaste.com/{paste-id}`

## Searching Pastes

Find your pastes quickly:

1. Go to **My Pastes**
2. Use the search bar to filter by:
   - Title
   - Content
   - Language
   - Date created
