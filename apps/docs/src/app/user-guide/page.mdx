# User Guide

Welcome to the psti.io User Guide. This section will help you get started with psti.io and make the most of its features.

## What is psti.io?

psti.io is an enterprise-level, security-focused pastebin designed specifically for security professionals. It provides a secure platform for sharing code snippets, logs, and text with advanced privacy and security features.

## Key Features

- **Multiple Privacy Levels** - Public, unlisted, private, and password-protected pastes
- **End-to-End Encryption** - AES-256-GCM encryption for sensitive content
- **Burn After Read** - Self-destructing pastes that delete after being viewed
- **Syntax Highlighting** - Support for 100+ programming languages
- **Expiration Dates** - Automatic paste deletion after a specified time
- **Folders & Organization** - Organize your pastes into collections
- **Raw Viewer** - Clean, distraction-free paste viewing
